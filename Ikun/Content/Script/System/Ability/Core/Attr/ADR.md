# 属性系统常见需求
1. 派生属性
    每1点力量增加5点生命上限和2点攻击力
2. 临时属性
    在GAS中通常不会直接修改Health而是修改一个IncomingDamage的隐藏属性
    当检测到IncomingDamage变化时: 触发护盾抵扣->减伤->无敌判断->扣除Health->IncomingDamage清零
